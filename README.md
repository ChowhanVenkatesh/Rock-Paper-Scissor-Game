Game Logic

User selects one of three options: Rock, Paper, or Scissors.

The computer randomly selects one option. Rules to determine the winner:

Rock beats Scissors.

Scissors beats Paper.

Paper beats Rock.

If both choose the same option, it's a tie. Screenshots
